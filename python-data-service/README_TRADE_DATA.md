# AKShare交易数据说明

## 重要说明

**AKShare无法获取具体股民的买卖明细**（个人交易数据属于隐私，不会公开）

但AKShare可以提供以下**市场交易数据**：

## 可获取的交易数据类型

### 1. 逐笔成交数据（Tick Data）
- **数据内容**：每笔交易的成交价、成交量、买卖方向
- **函数示例**：
  - `stock_zh_a_tick_tx` - 腾讯逐笔成交
  - `stock_zh_a_tick` - 逐笔成交
  - `stock_tick_em` - 逐笔成交（东方财富）
- **用途**：分析买卖力量对比、交易活跃度

### 2. 分时成交数据（Minute Data）
- **数据内容**：每分钟的成交汇总（开盘价、收盘价、最高价、最低价、成交量）
- **函数示例**：
  - `stock_zh_a_minute` - 分时数据
  - `stock_zh_a_minute_em` - 分时数据（东方财富）
- **用途**：分析日内价格走势、成交量变化

### 3. 买卖盘口数据（Order Book）
- **数据内容**：买一、买二、买三...卖一、卖二、卖三...的委托价格和数量
- **函数示例**：
  - `stock_zh_a_bid_ask_em` - 买卖盘口
  - `stock_zh_a_spot_em` - 实时行情（包含买卖盘）
- **用途**：分析市场深度、支撑阻力位

### 4. 大单交易数据
- **数据内容**：大资金的买卖情况（通常指单笔成交额较大的交易）
- **函数示例**：
  - `stock_lhb_em` - 龙虎榜（大单交易）
  - `stock_bid_ask_em` - 买卖盘分析
- **用途**：分析主力资金动向

### 5. 资金流向数据
- **数据内容**：主力资金、散户资金的流入流出情况
- **函数示例**：
  - `stock_fund_flow_individual` - 个股资金流向
  - `stock_fund_flow_em` - 资金流向
- **用途**：分析资金流向、判断主力意图

## 测试方法

运行测试脚本：
```bash
python test_trade_data.py 300474
```

## 注意事项

1. **交易时间限制**：某些实时数据仅在交易时间内可用（9:30-11:30, 13:00-15:00）
2. **数据延迟**：免费数据源通常有15分钟延迟
3. **VIP权限**：某些详细数据可能需要VIP权限或付费接口
4. **隐私保护**：具体投资者的交易明细不会公开

## 替代方案

如果需要更详细的交易数据，可以考虑：
1. **付费数据服务**：Wind、同花顺、东方财富等专业数据服务
2. **券商API**：部分券商提供Level-2行情数据
3. **量化平台**：聚宽、米筐等量化平台提供更详细的数据

## 实际应用

虽然无法获取具体股民的买卖数据，但可以通过以下方式分析市场：
- **逐笔成交** → 分析买卖力量对比
- **大单数据** → 追踪主力资金动向
- **资金流向** → 判断市场情绪
- **盘口数据** → 分析市场深度和支撑阻力
