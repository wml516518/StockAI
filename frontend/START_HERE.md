# 🚀 快速开始

## ✅ Node.js 已安装

Node.js v24.11.0 和 npm v10.8.2 已成功安装！

## 📦 依赖已安装

所有前端依赖已成功安装。

## 🎯 下一步

### 1. 确保后端服务正在运行

在另一个 PowerShell 窗口中启动后端 API：

```powershell
cd D:\Demo\StockAnalyse\src\StockAnalyse.Api
dotnet run
```

后端服务将在 `http://localhost:5000` 启动。

### 2. 启动前端开发服务器

在当前 PowerShell 窗口中运行：

```powershell
npm run dev
```

前端开发服务器将在 `http://localhost:5173` 启动。

### 3. 访问应用

打开浏览器访问：**http://localhost:5173**

## 💡 提示

- 前端开发服务器会自动代理 `/api` 请求到后端
- 修改代码后会自动刷新页面（HMR）
- 按 `Ctrl+C` 可以停止开发服务器
